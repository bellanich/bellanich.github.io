Bella Nicholson, Aoibhinn Reddington

Abstract

I delivered a guest lecture at TU Delft’s Bachelor’s Machine Learning course. I unpacked core Machine Learning Engineering theory and then showcased its practical application through a case study. This case study detailed how I improved an established Dutch tech company’s MLOps maturity, demonstrating the real-world impact of these concepts.
